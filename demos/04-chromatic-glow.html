Chromatic Glow

This technique creates a subtle multi-color glow around the content area using blur filters and blend modes. It's reminiscent of aurora borealis without being obvious about it.

The Effect

Look at the edges of the content area—notice a very subtle colored haze? It combines your accent colors (cyan #63bbb9 and coral #f57a59) in a radial gradient that's heavily blurred and uses screen blend mode.

  • Uses both accent colors in gradient
  • 40px blur creates soft, diffuse light
  • Screen blend mode for ethereal quality
  • Positioned around content, not full viewport
  • Creates aurora-like atmosphere

The effect is barely visible on the dark background, which is intentional—it should feel like ambient light rather than a spotlight.

The Code

.container::before { content: ''; position: absolute; background: radial-gradient( ellipse at center, rgba(99, 187, 185, 0.15) 0%, rgba(245, 122, 89, 0.08) 50%, transparent 70% ); filter: blur(40px); mix-blend-mode: screen; opacity: 0.5; }

Why It Works

The screen blend mode is key—it makes dark areas darker and light areas lighter, similar to how light actually behaves. Combined with blur, it creates soft, glowing edges that feel more like atmospheric lighting than a design effect.

The two-color gradient (cyan to coral) mirrors the aurora borealis color range and ties your design system together by using both accent colors simultaneously.

Best For

If you want your aurora/astronomical theme to be more literal. This adds ambient color that changes character across the content area. More noticeable than other techniques but still subtle. Works best on wider viewports where the glow has space to diffuse.

Performance Note

The blur filter does require GPU compositing. On modern devices this is negligible, but older/mobile devices may see minor performance impact. Test before deploying site-wide.
