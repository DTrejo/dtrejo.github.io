Hero Fade

This technique uses a full-viewport background that gradually fades from a starfield at the top to solid color as you scroll down. It's the most dramatic option—perfect for article pages.

Note: This demo uses an actual Milky Way photograph from Unsplash. You can swap it for any astronomical image—aurora borealis, nebula, etc.

The Effect

The background starts with a visible Milky Way photograph at the top, then gradually fades to your solid #013250 color as you scroll down. This creates a sense of depth and ties into your astronomical theme directly.

  • Fixed positioning keeps background stable while content scrolls
  • Linear gradient mask fades image to color (0% → 30% → 60% → 100%)
  • Works with any image: Milky Way, aurora, nebula
  • Most impactful on article pages
  • Optional: add parallax scrolling for depth
  • Uses actual Unsplash photo (~150KB, loads quickly)

The Code

.hero-bg { position: fixed; background: linear-gradient(180deg, transparent 0%, rgba(1, 50, 80, 0.3) 30%, rgba(1, 50, 80, 0.7) 60%, #013250 100% ), url('https://images.unsplash.com/photo-...') center top / cover no-repeat; } /* Alternative with aurora: */ /* url('https://images.unsplash.com/photo-1529963183134...') */

Why It Works

This approach is used by Medium, Stripe docs, and many editorial sites. The image creates immediate visual interest and emotional connection, while the fade ensures it doesn't compete with content as the user reads.

For your site, using actual astronomy photos is literal rather than metaphorical—your background shows the same Milky Way aesthetic you use in social images. It creates thematic consistency across your brand.

Implementation Notes

This demo uses a Milky Way photo, but you can easily swap to aurora or other astronomical imagery:

  • Milky Way: photo-1515951834549-4172b316de7e (current)
  • Aurora: photo-1529963183134-61a90db47eaf
  • Use WebP format for smaller file size (40-80KB target)
  • Adjust imgix params: ?fit=crop&h=1080&w=1920&q=80
  • Consider different images per article or category

Parallax Enhancement

You could add subtle parallax scrolling (background moves at 0.5x speed) using JavaScript. This creates a sense of depth as the user scrolls. Let me know if you'd like me to add this to the demo.

Best For

Article pages where you want to make an impression. This is your most dramatic option—it puts astronomy front and center. Not recommended for the homepage or index pages where simplicity is better, but perfect for long-form content.
