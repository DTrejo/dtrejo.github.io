Grain + Gradient

This is my top recommendation. It combines monochromatic noise (film grain) with a subtle three-color vertical gradient. The result is analog warmth with digital sophistication.

The Effect

Two layers working together:

  1. Vertical gradient — Lighter blue at top (#013e5f) → your base (#013250) at middle → slightly darker (#012e4a) at bottom. Creates subtle vertical rhythm.
  2. Film grain texture — SVG noise pattern at 15% opacity with overlay blend mode. Adds tactile quality like paper or canvas.

Zoom your browser to 150-200% and look at any solid area—you'll see the grain clearly. At normal zoom, it's almost invisible but adds richness you can feel.

The Code

/* Gradient layer (::before) */ body::before { background: linear-gradient(180deg, #013e5f 0%, #013250 50%, #012e4a 100% ); } /* Grain layer (::after) */ body::after { background-image: url('data:image/svg+xml...'); background-size: 128px 128px; opacity: 0.15; mix-blend-mode: overlay; }

Why It Works

Film grain is neurologically interesting—our brains interpret it as "analog" and "premium" because we associate it with film photography and high-end printing. Digital screens are perfectly smooth by default, so adding grain creates unexpected texture.

The three-color gradient prevents the background from feeling flat without being obvious. The colors are close enough in value that it reads as subtle shading rather than distinct bands.

Together, these create what designers call "material quality"—the background feels like it has substance rather than being pure digital flatness.

Design Precedent

This technique is used by:

  • Linear — Gradient + grain on their dark mode
  • Figma — Subtle noise on canvas backgrounds
  • Stripe — Grain texture on hero sections
  • Are.na — Textured backgrounds for visual richness

Performance

The grain is an inline SVG data URI (no HTTP request) and tiles at 128×128px, making it extremely lightweight. The gradient is pure CSS. Combined file size impact: ~0.5KB. No JavaScript, no external images, no performance hit.

Variants to Consider

You could adjust:

  • Grain opacity: 10-20% range (currently 15%)
  • Grain size: 64px (more visible) or 256px (finer)
  • Blend mode: Try soft-light instead of overlay for softer effect
  • Gradient colors: Could add hint of your accent colors

Best For

If you want sophistication that reveals itself on closer inspection. This combines the best aspects of several other demos: texture (like layered depth), vertical interest (like vignette), and material quality that's unique to this approach.

It's subtle enough for every page, sophisticated enough to feel intentional, and adds warmth without astronomical imagery. If you only implement one enhancement, make it this one.
