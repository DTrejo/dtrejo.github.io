Meteor Streaks

This technique creates occasional shooting star effects—fast diagonal streaks that appear every 30-60 seconds. They're extremely subtle: blink and you'll miss them.

The Effect

Watch the background for 30-60 seconds. You'll see brief flashes of light streaking diagonally across the screen. Each meteor:

  • Appears at random position in upper area
  • Falls diagonally (down and to the right)
  • Lasts only 1-2 seconds total
  • Has a glowing trail that fades
  • Appears every 30-60 seconds (randomized)

The timing mimics real meteor showers—you're not constantly watching for them, but when one appears, it catches your peripheral vision.

The Code

.meteor::before { content: ''; width: 80px; height: 2px; background: linear-gradient(90deg, white, transparent); } @keyframes meteor-fall { 0% { opacity: 0; transform: translate(0, 0); } 5% { opacity: 1; } 100% { opacity: 0; transform: translate(300px, 300px); } } // JavaScript spawns meteors at random intervals function createMeteor() { const meteor = document.createElement('div'); meteor.className = 'meteor'; // Random position, random timing... }

Why It Works

Shooting stars create moments of delight. In nature, seeing a meteor is special—it's unpredictable and fleeting. By recreating this digitally:

  • Surprise factor: Random timing prevents habituation
  • Peripheral vision: Movement catches eye without demanding focus
  • Wishful thinking: Cultural association with making wishes
  • Real phenomenon: Everyone has seen real shooting stars

The key is restraint—too many meteors become annoying. Real meteor showers average 1-2 per minute during peak activity. We match that pace.

Accessibility

Respects prefers-reduced-motion by hiding all meteors completely. For users with vestibular disorders, sudden motion can cause discomfort—better to disable entirely than try to slow it down.

Variations

You could:

  • Change color to cyan or coral for variety
  • Add particle trail that dissipates
  • Create "meteor shower" mode with burst of several
  • Angle meteors in different directions
  • Make speed variable (faster = brighter in real life)

Best For

If you want moments of surprise and delight. This is more playful than other astronomical techniques—it adds personality and whimsy. Best for sites where occasional animation won't distract from serious reading, or where you want to reward long-time visitors who might catch multiple meteors.
